# Nanoviolin

**work in progress**

The Vaccine Investigation and OnLine Information Network (VIOLIN) is a major vaccine information and knowledge base. This work introduces an informatics implementation to integrate and align VIOLIN for the [nanopublication](https://nanopub.net/) network - a semantic micropublication platform to support the linking and dissemination of scientific assertions.
