# Nanoviolin

**work in progress**

The Vaccine Investigation and OnLine Information Network (VIOLIN) is a major vaccine information and knowledge base. This work introduces an informatics implementation to integrate and align VIOLIN for the [nanopublication](https://nanopub.net/) network - a semantic micropublication platform to support the linking and dissemination of scientific assertions.


## Usage

The work presented is in experimental stage. The easy way to experiment with software is to open it up with [NetBeans IDE](https://netbeans.apache.org/front/main/index.html). You will also need to install a OpenJDK distribution (e.g. Temurin).

- clone the repo
- open with NetBeans IDE
- either click Run Project (F6) or run the file "Nanoviolin.java"

Also, a Sqlite database will be created in the project folder. You may need to open it with SQLite viewer (e.g. DB Browser for Sqlite).
